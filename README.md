# 🏆 FIFA World Cup 2026 Predictor

AI-powered match predictor with historical data, squad comparison, and head-to-head analysis.

## Features
- Live World Cup 2026 matches & standings
- Player stats, team stats, top scorers
- **Match Predictor** – win probability using historical data, head-to-head, squad quality, and current form
- Multi-language support (English, Hindi, Telugu)
- AI assistant (ADK Agent, Ollama, OpenAI BYOK)
- CI/CD with quality checks

## Quick Start
```bash
git clone https://code.swecha.org/Ayushchavan/Fifa-2026-predictor.git
cd Fifa-2026-predictor
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## Tech Stack
- Python, Streamlit, Pandas, Plotly
- Google ADK, Ollama, OpenAI
- GitLab CI/CD, pre-commit hooks
