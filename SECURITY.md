# Security Policy

## Reporting a Vulnerability
Please report security issues to ayushchavan1411@gmail.com.

We will respond within 48 hours and work to patch the issue.

## Supported Versions
We support the latest version of the `main` branch.

## Best Practices
- Do not commit API keys or tokens.
- Use environment variables for secrets (see `.env.example`).
- Run `bandit` locally to detect security issues.
